Powerful Features

Everything you need for nanoscale optical calculations

Mie Scattering

Accurate Mie theory calculations for spherical nanoparticles. Calculate Q_sca, Q_abs, and Q_ext with complex refractive indices.

Spectral Analysis

Full wavelength scanning from 300-800 nm. Interactive plots with real-time updates and zoom capabilities.

Periodic Table

Built-in database of optical properties for 13 common elements at 550 nm wavelength.

Multilingual

Full support for English and Spanish interfaces with instant language switching.

High Performance

Built with Rust for native performance. Near-instant calculations even for complex spectra.

Educational

Detailed tooltips explain the physics behind each parameter. Perfect for students and researchers.
